報名程序

一、報名方式一律採取網路報名，且以 校系 為單位接受報名。

二、請各校系於 此處 下載報名表，填妥 團體賽以及個人賽 欲報名之 項目與隊數 及兩位聯絡人的電話。報名表裡包括團體賽及個人賽，填寫方式如以下範例（圖一）。( 註：本範例填寫之資料僅供示範用，非本系報名之正確數量。 )

（圖一）

三、請各校3/21(二) 前寄回至此次大物盃的 聯絡信箱 ，並於 3/21 (二) 下午5點 前完成所有賽事的報名費匯款 ( 註：匯款帳號如上 ) ，逾期不候。為方便作業，報名以收到匯款為主，未匯款視同未報名。 我們收到報名費後將以電話方式通知貴系聯絡人確定報名完成。若匯款後並未接到通知請主動聯絡我們，並保留單據以供日後查詢

四、完成報名之校系請在 3/25 (六) 以前再以網路繳交 球員資料與學生證正反面影本。

五、球員資料的統一格式請由 此處 下載，填寫時請將下方工作表（圖二）調整成貴校的實際報名隊伍狀況 ( 複製工作表) 。球員名單有 "*" 者 ( 隊長及兩位隊員 ) 請務必填寫「手機」一欄作為大會緊急聯絡通知用；球員中為實習老師身分者請在備註欄上加註。
