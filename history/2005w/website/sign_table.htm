報名方式一律採取網路報名，且以 校系 為單位接受報名。

請各校系於 此處 下載報名表，填妥 團體賽 ( 個人賽尚不需確定 ) 欲報名之 項目與隊數 及兩位聯絡人的電話後，於 10/11 (二) 前寄回至此次大物盃的 專用信箱 ，並於 10/13 (四) 下午 5 點 前完成團體賽的報名費匯款 ( 匯款帳號如上 ) ，逾期不候。為方便作業，報名以收到匯款為主，未匯款視同未報名。

我們收到報名費後將以電話方式通知貴系聯絡人確定報名完成。若匯款後並未接到通知請主動聯絡我們，並保留單據以供日後查詢。

完成報名之校系請在 10/18 (二) 以前再以網路繳交 球員資料與學生證正反面影本 ，連同個人賽的參賽名單，再於 10/21 (五) 下午 5 點 前完成個人賽的報名費匯款。 ( 個人賽報名費如上所列 )
